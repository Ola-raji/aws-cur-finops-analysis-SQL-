# Sample data

This folder is reserved for an anonymised CUR sample (`cur-sample.csv`) plus its `manifest.json`. The sample is generated by `scripts/anonymise.py` from a private source CUR; the script substitutes the AWS account ID and payer name with placeholders consistently across direct columns, ARNs, and bucket names.

The sample is not yet committed. Until it is, the SQL in `sql/` is reproducible against any CUR 2.0 export loaded into `staging.cur_line_items` with the canonical column names.
